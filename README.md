# Component rollup config

> Extensible rollup configuration to build OVH components

## Install

```sh
yarn add -D @ovh-ux/component-rollup-config
```

## Usage

```js
import config from 'component-rollup-config';

export default config({
  input: 'src/index.js'
})
```

## Test

```sh
yarn test
```
